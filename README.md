### üí° **My Learning Methodology (How to Learn)**

My 15-day strategy is divided into three focused phases:

#### Phase 1: Foundations (Day 1 - 5)
* **Focus:** Core Syntax, Data Types (Int, Float, Boolean), Variables, Operators, and Control Flow (If/Else, Loops).
* **Practice Goal:** Completing all basic exercises on W3Schools and solving simple arithmetic problems.

#### Phase 2: Data Structures & Functions (Day 6 - 10)
* **Focus:** Understanding and implementing Python's primary built-in data structures:
    * **List** (Arrays)
    * **Tuple**
    * **Dictionary** (Hash Maps)
    * **Set**
* **Practice Goal:** Writing and using custom functions, parameter passing, return values, and list comprehension.

#### Phase 3: OOP & Practical Application (Day 11 - 15)
* **Focus:** Object-Oriented Programming (Classes, Objects, Inheritance), File Handling (I/O), and Introduction to basic Modules (e.g., `random`, `math`).
* **Practice Goal:** Building small, practical command-line applications (e.g., a simple calculator or file reader) to solidify learning.
